#gir
Github Issue Ripper

Objective
---------
Rip all the issues from a set of repositories into individual files

Dependencies
------------
gir makes use of :
<ul>
<li>Ruby 2.1.5 although eariler versions will most certianly work</li>
<li>Octokit.rb (a ruby gem for parsing github repositories</li>
</ul>
